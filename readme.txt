Objetivo: Crear un menú de hamburguesa siguiendo el modelo mostrado en la pantalla. 
El proyecto consta de dos partes:
La primera parte és un banner, en el que hay un botón 3D con efecto de desplazamiento. Al hacer clic en el icono del menú Hamburguesa, se nos dirigirá a la segunda parte del proyecto. Será una barra de navegación con dos navegaciones diferentes. 
El proyecto utiliza HTML, CSS y JavaScript.
